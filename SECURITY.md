# Security Policy

## Supported Versions

Security updates are provided for the latest released version of Folder Structure Generator.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report security concerns by emailing ujjawalpoudel@gmail.com with:

- A description of the vulnerability.
- Steps to reproduce the issue.
- The affected version.
- Any suggested fix or mitigation, if known.

You can expect an initial response within 7 days. Confirmed vulnerabilities will be fixed as soon as practical and released with clear notes in the changelog.
