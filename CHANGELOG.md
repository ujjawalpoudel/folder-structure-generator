# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Added a `folder-structure` command-line interface.
- Added deterministic, sorted tree output.
- Added options for ignored folders, hidden files, directory-only output, and maximum depth.
- Added standard-library tests for the generator.
- Modernized package metadata in `pyproject.toml`.

## 0.0.2

- Initial public package metadata.
